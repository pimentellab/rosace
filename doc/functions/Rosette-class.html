Rosette-classR Documentation

The Rosette Class

Description

A Rosette object is a container for inferring summary statistics from a AssayGrowth of a Rosace experiment.

Slots

score.df

A data.frame with columns score, pos, mut, ctrl, (var_group), (mut_group)

mut.label

A data.frame with columns mut, mut_group

var.dist

A data.frame with columns var_group, mean, sd, count

mut.var.alpha

A data.frame with columns mut_group, var_group, alpha, count

disp

Dispersion of raw count (sequencing step)

disp.start

Dispersion of starting variant library

Gvar

Number of variant groups

Gmut

Number of mutant groups

rounds

Number of rounds

project.name

A character string indicating the name of the project
