PlotScoreHistR Documentation

Plot histogram of scores in Rosette object

Description

Plot histograms with variant group label and/or mutant group label

Usage

PlotScoreHist(object, var.group = FALSE, mut.group = FALSE)

Arguments

object

An Rosette object

var.group

If TRUE, plot histograms with variant group label

mut.group

If TRUE, plot histograms with mutant group label

Value

A ggplot object
