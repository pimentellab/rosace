GenMutLabelR Documentation

Generate mutant group label in Rosette object

Description

Generate mutant group label in Rosette object

Usage

GenMutLabel(object, Gmut, hclust, save.plot = FALSE, save.dir = NULL)

Arguments

object

An Rosette object

Gmut

Number of mutant groups

hclust

Input from function HclustMutant. If not given, will call HclustMutant.

save.plot

If TRUE, save the plot to the directory save.dir

save.dir

The directory to save the plot

Value

An Rosette object with mut.label and Gmut
