PMVCountDistR Documentation

Infer distribution for number of variants for each variant and mutant group label at each position

Description

Fit a Dirichlet distribution for number of variants in (mutant group, variant group) at each position

Usage

PMVCountDist(object, pos.missing = 0.2)

Arguments

object

A Rosette object

pos.missing

Percentage of variants missing at each position allowed

Value

A Rosette object with mut.var.alpha
